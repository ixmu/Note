## 特性
- 最小化的请求数，最大化的速度。
- 拥有详尽的代码注释，方便二次开发。
- 结合实际使用场景为每一种支付方式独立定制的展示样式。
- 高度可定制化，页面上一切内容均可通过简单的 Javascript 代码更改。无需任何编程技巧。
- 所有静态文件均默认使用开源 CDN [jsDelivr](https://www.jsdelivr.com) 分发。
